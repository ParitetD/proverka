## 📋 Описание проекта

Это **единый проект**, объединяющий все практические задания по Frontend-разработке (HTML/CSS + React) для стажировки в Axoft IT Solutions.

Вместо создания отдельных папок для каждого задания, все задания интегрированы в **одно современное React-приложение** с:
- 🎨 Красивой навигацией между заданиями
- ⚡ Живыми демо всех компонентов
- 🔥 Креативным дизайном с градиентами и анимациями
- 📱 Полной адаптивностью для мобильных устройств
- 🖼️ Реальными фотографиями из Unsplash

- ✅ Media queries для адаптивности
- ✅ Гамбургер-меню для мобильных (<768px)

## 🚀 Запуск проекта

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откроется браузер на `http://localhost:5173`

### Сборка для продакшена

```bash
npm run build
```
